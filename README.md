# Mac Setup Script

My setup.

## Pre

1. System Settings -> General -> Software Update

## How to run

1. **Clone this repo** (or download the setup.sh script):

```bash
cd ~
git clone https://github.com/aaronjbaptiste/mac-setup.git
cd mac-setup
chmod +x setup.sh
./setup.sh
```

2. Restart Mac
